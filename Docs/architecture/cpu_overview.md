# CPU Overview

The 16-Bit FPGA Processor is a custom accumulator-based processor designed for educational purposes and implemented entirely in Verilog.

The processor executes a custom instruction set developed specifically for this architecture and is capable of running programs generated by the accompanying assembler and C-like compiler.

## Design Goals

- Simple hardware implementation
- Easy to understand instruction set
- Modular Verilog design
- FPGA implementation
- Support for a complete software toolchain

## Processor Features

- 8-bit datapath
- 16-bit program counter
- Custom accumulator architecture
- Eight general-purpose registers
- Eight JAL registers
- Hardware interrupt support
- Memory-mapped I/O
- VGA text output
- Seven-segment debugging output

## Software Toolchain

Programs execute through the following compilation flow:

```
C-like Language
        │
        ▼
Compiler
        │
        ▼
Assembly
        │
        ▼
Assembler
        │
        ▼
Machine Code
        │
        ▼
FPGA Processor
```

The compiler, assembler, and processor were developed together and share a common instruction set architecture.
