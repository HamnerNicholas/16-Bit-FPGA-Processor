# RTL Schematics

This directory contains RTL viewer diagrams generated by Intel Quartus after synthesis.

These diagrams represent the synthesized hardware implementation of each module and are intended as supporting documentation alongside the architectural diagrams found elsewhere in the repository.

The schematics are grouped by subsystem:

- **CPU_MAIN** – Complete processor integration.
- **processor** – Core datapath modules.
- **control** – Instruction decoding and control logic.
- **io** – Display, VGA, and peripheral hardware.

Note that Quartus may optimize portions of the design during synthesis. As a result, these schematics may differ slightly from the original Logisim design while remaining functionally equivalent.
