# Memory Map

The processor separates program memory, global memory, and interrupt vectors into independent memory spaces.

## Program Memory

Stores executable machine instructions generated by the assembler.

```
0x0000 - 0xFFFF
```

## Global Memory

Stores global variables and static data.

```
0x0000 - 0x00FF
```

## Interrupt Vector Table

Eight interrupt vectors are maintained separately from program memory.

```
INT0
INT1
INT2
INT3
INT4
INT5
INT6
INT7
```

## Future Expansion

The architecture reserves additional address space for future peripherals including:

- Keyboard controller
- VGA text buffer
- Timer
- Additional memory-mapped devices
