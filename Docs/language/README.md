# Language Documentation

This section contains the programming language documentation for the 16-Bit CPU software ecosystem.

Two programming languages are supported:

- **High-Level Language** – A lightweight C-inspired language that is compiled into assembly.
- **Assembly Language** – The native instruction set executed by the 16-Bit CPU.
