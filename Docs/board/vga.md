# VGA Output

The current FPGA implementation provides VGA output through a dedicated VGA controller.

The display uses a 4-bit RGB interface.

---

## Signals

| Signal | Width |
|---------|------:|
| VGA_R | 4 bits |
| VGA_G | 4 bits |
| VGA_B | 4 bits |
| VGA_HS | 1 |
| VGA_VS | 1 |

---

## Pin Assignments

### Red

| Signal | FPGA Pin |
|---------|----------|
| VGA_R[0] | PIN_AA1 |
| VGA_R[1] | PIN_V1 |
| VGA_R[2] | PIN_Y2 |
| VGA_R[3] | PIN_Y1 |

### Green

| Signal | FPGA Pin |
|---------|----------|
| VGA_G[0] | PIN_W1 |
| VGA_G[1] | PIN_T2 |
| VGA_G[2] | PIN_R2 |
| VGA_G[3] | PIN_R1 |

### Blue

| Signal | FPGA Pin |
|---------|----------|
| VGA_B[0] | PIN_P1 |
| VGA_B[1] | PIN_T1 |
| VGA_B[2] | PIN_P4 |
| VGA_B[3] | PIN_N2 |

### Synchronization

| Signal | FPGA Pin |
|---------|----------|
| VGA_HS | PIN_N3 |
| VGA_VS | PIN_N1 |

---

## Current Usage

The VGA interface is currently used for framebuffer output generated by the processor.

Future revisions may include hardware-accelerated graphics primitives and text rendering.
